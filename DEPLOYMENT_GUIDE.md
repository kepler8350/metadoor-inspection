# 🚀 메타도어 웹서비스 - 완전 배포 가이드

## ✨ **완성된 기능**

### **1️⃣ 모바일 앱** (`/`)
- 로그인 화면
- 구 선택 화면 (부산 15개 구)
- 메타도어 설치 위치별 리스트
- 점검 입력 폼

### **2️⃣ 관리자 페이지** (`/admin`)
- **로그인**: `/admin/login`
- **대시보드**: `/admin` (인증 필요)
- **통계**: 총 점검 수, 정상/요주의/이상 카운트
- **점검 기록 테이블**: 실시간 갱신

---

## 📱 **모바일 앱 사용법**

### **URL**: `https://metadoor-inspection-production.up.railway.app/`

1. **로그인**
   - 아이디: `admin`
   - 비밀번호: `admin123`

2. **구 선택**
   - 왼쪽: 부산 15개 구 버튼
   - 오른쪽: 선택한 구의 메타도어 설치 위치 (3개)

3. **점검 입력**
   - 점검 항목 선택 (외벽, 지붕, 창호, 바닥, 내벽, 기타)
   - 조사 내용 입력
   - 점검 상태 선택 (정상, 요주의, 이상)
   - 완료 버튼으로 저장

## 📊 **API 엔드포인트**

### **점검 저장**
```bash
POST /api/inspection
Content-Type: application/json

{
  "district": "사상구",
  "location": "삼락공원",
  "item": "외벽",
  "content": "페널 박인",
  "status": "정상"
}

Response: { "success": true }
```

### **점검 기록 조회** (관리자 인증 필요)
```bash
GET /api/inspections
Response: [
  {
    "id": 1,
    "district": "사상구",
    "location": "삼락공원",
    "item": "외벽",
    "content": "페널 박인",
    "status": "정상",
    "created_at": "2024-05-13 14:30:00"
  }
]
```

- **프로젝트**: `refreshing-connection`
- **서비스**: `metadoor-inspection`
- **URL**: `https://metadoor-inspection-production.up.railway.app`

### **로컬 테스트**
```bash
# 1. 저장소 클론
git clone https://github.com/kepler8350/metadoor-inspection.git
cd metadoor-inspection

# 2. 의존성 설치
pip install -r requirements.txt

# 3. 앱 실행
python app.py

# 4. 브라우저에서 접속
# 모바일: http://localhost:8080/
# 관리자: http://localhost:8080/admin
```
